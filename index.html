Recomendações fixas
Zona cinzenta: FC média 134–140 bpm nos seus treinos. Reduza o ritmo até conseguir falar frases inteiras com conforto — isso é Z2 real (116–139 bpm).
Natação (sexta): Dia ideal — após bloco ter+qui e antes do sábado longo. Foque em técnica de bracada e rolamento, não em distância bruta.
Double threshold (ter+qui): Bike na terça, corrida na quinta. Este é o núcleo do método norueguês — proteja essas sessões acima de tudo.
Calor em Haynes: Em dezembro na Flórida, mesma FC = pace mais lento. Treine sempre por FC, nunca por pace. No dia da prova, saia conservador na bike.

Passo a passo — GitHub Pages (5 minutos)
1
Criar conta no GitHub
Acesse github.com e crie uma conta gratuita se ainda não tiver. O plano gratuito já inclui o GitHub Pages.
2
Criar um novo repositório
Clique em New repository. Nomeie como ironman703. Marque como Public. Clique em Create repository.
3
Fazer upload do arquivo HTML
Na página do repositório, clique em Add file → Upload files. Arraste o arquivo ironman703_dashboard.html que você baixou. Renomeie para index.html antes de subir (ou renomeie após o upload). Clique em Commit changes.
4
Ativar o GitHub Pages
Vá em Settings → Pages (menu lateral). Em Source, selecione Deploy from a branch. Em Branch, selecione main e pasta / (root). Clique em Save.
5
Acessar seu dashboard
Após ~2 minutos, seu dashboard estará disponível em:
https://seuusuario.github.io/ironman703
Salve nos favoritos e adicione à tela inicial do celular!
6
Adicionar à tela inicial (iPhone)
Abra o link no Safari → toque no ícone de compartilhar → Adicionar à Tela de Início. O dashboard vai aparecer como um app.
